Clustering.jl package

Clustering.jl is a Julia package for data clustering. It covers the two aspects of data clustering:

  • Clustering Algorithms: K-means, K-medoids, Affinity propagation, DBSCAN etc.
  • Clustering Comparison & Evaluation: cross-tabulation, variational and mutual information, intrinsic clustering quality indices, such as silhouettes, etc.
